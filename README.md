## Клиентская часть клона TJournal.

# Technology:

- NextJS / React
- TypeScript
- Redux / Redux Saga
- EditorJS
- React Hook Form / Yup
- Material UI
- SCSS / CSS-modules

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
